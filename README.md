# Hướng Dẫn Cài Đặt và Xây Dựng Ứng Dụng Flutter

## Bước 1: Cài Đặt Flutter

Để bắt đầu, bạn cần cài đặt Flutter phiên bản 3.35.4. Vui lòng làm theo hướng dẫn tại [đây](https://docs.flutter.dev/get-started/quick).

### Các bước cài đặt:

1. Tải xuống Flutter SDK từ trang chính thức.
2. Giải nén tệp tải về vào thư mục bạn muốn lưu trữ Flutter.
3. Thêm Flutter vào biến môi trường `PATH`.
4. Chạy lệnh `flutter doctor` để kiểm tra các yêu cầu cần thiết và cài đặt bất kỳ phần mềm phụ trợ nào còn thiếu.

## Bước 2: Chạy và Xây Dựng Ứng Dụng

### Để debug ứng dụng:

Chạy lệnh sau trong terminal:

```
flutter run
```

### Để xây dựng file APK:

Chạy lệnh sau trong terminal:

```
flutter build apk --release
```
